# 🐍 Snake Game (Python)

A classic Snake Game built with Python. Control the snake, eat food to grow longer, and try to achieve the highest score without colliding with the walls or yourself.

## Features

* Smooth snake movement
* Score tracking system
* Random food generation
* Collision detection
* Game Over screen
* Simple and clean Python implementation

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/snake-game.git
cd snake-game
```

### 2. Install dependencies

```bash
pip install pygame
```

### 3. Run the game

```bash
python main.py
```

## Controls

| Key | Action     |
| --- | ---------- |
| ↑   | Move Up    |
| ↓   | Move Down  |
| ←   | Move Left  |
| →   | Move Right |

## How to Play

1. Start the game.
2. Use the arrow keys to control the snake.
3. Eat food to increase your score and snake length.
4. Avoid hitting the walls.
5. Avoid colliding with your own body.
6. Try to achieve the highest score possible.
